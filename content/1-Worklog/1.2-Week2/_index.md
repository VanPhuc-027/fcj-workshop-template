### Week 2 Achievements:

* Mastered foundational knowledge of AWS VPC and its core components: Subnets (Public/Private), Internet Gateway (IGW), NAT Gateway, and Route Tables.

* Clearly understood network security mechanisms within a VPC using Security Groups (stateful) and Network ACLs (stateless).

* Differentiated and learned how to apply scalable hybrid connectivity solutions: **VPN (Site-to-Site)**, AWS Direct Connect, and the centralized router AWS Transit Gateway.

* Grasped the operational principles of Elastic Load Balancing (ELB) in distributing application traffic.
