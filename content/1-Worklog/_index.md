**On this page**, I summarize my complete Worklog throughout the internship program and project implementation on AWS. The program was completed within **12 weeks** (approximately 3 months), detailing the full journey from foundational theory and weekly hands-on labs to project research and final real-world deployment.

The detailed task breakdown for each week is summarized below:

**Week 1:** [Get familiar with AWS and foundational AWS services, complete Module 01, and perform related hands-on labs.](1.1-week1/)

**Week 2:** [Study Module 02 on Cloud Compute fundamentals & Amazon EC2 services.](1.2-week2/)

**Week 3:** [Study Module 03 on Amazon VPC, Networking & Security Groups.](1.3-week3/)

**Week 4:** [Study theory & hands-on labs for Module 04 on Amazon S3, Hybrid Storage & Storage Gateway.](1.4-week4/)

**Week 5:** [Study Module 05 on Identity & Access Management and System Security with AWS IAM, Cognito, Organizations & Security Hub.](1.5-week5/)

**Week 6:** [Study Module 06 on AWS Database Services (RDS, Aurora, Redshift, ElastiCache) & Data Migration using AWS DMS.](1.6-week6/)

**Week 7:** [Practice Big Data Analytics, Data Lakes, Amazon DynamoDB & AWS Glue; review core knowledge and research architectural blogs.](1.7-week7/)

**Week 8:** [Complete remaining Module 07 hands-on labs, review previous practical labs and theoretical concepts.](1.8-week8/)

**Week 9:** [Research Architecture Blogs & finalize the project topic: "Automated Playwright Testing System Containerized with Docker on AWS".](1.9-week9/)

**Week 10:** [Draft, collect peer/mentor feedback, and optimize the System Architecture Diagram.](1.10-week10/)

**Week 11:** [Finalize Event-Driven Architecture (EDA), delegate team tasks, and configure Docker/Amazon ECR.](1.11-week11/)

**Week 12:** [Deploy Amazon ECS Cluster, integrate AI/CloudWatch/S3, debug system issues, and finalize the Workshop documentation.](1.12-week12/)
